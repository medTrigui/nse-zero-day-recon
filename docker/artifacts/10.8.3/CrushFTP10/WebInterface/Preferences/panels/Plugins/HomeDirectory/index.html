HomeDirectory Plugin Help

Simple plugin that just generates the users home directory when they first login. This way you could use say the SQL plugin and setup a user. When they actually login, a new folder could be created for them to upload into.

How to use:

Set a location where users home folders will be created at. Use the browse button to pick the location.

Specify the permission they will have in that folder. In general you will give them full permissions here as this folder is probably empty. This way they can then login, and upload files to it. If the folder doesn't exist, it will be created for them.

To test, create a new user via the user manager. Don't setup any directory items in the "user's stuff (VFS)" pane. Then login with that user and look for the new folder that just got created. For the time/date format on a unique folder, search google for 'SimpleDateFormat'.
